# Techboard — Landing Page

Este projeto é uma landing page simples desenvolvida usando HTML e CSS.

Breve descrição:

- O projeto contém a página principal em `index.html` e os estilos em `CSS/style.css`.
- A pasta `img/` contém as imagens utilizadas na página.
- Foi desenvolvido como um site estático para apresentação, praticando layout, responsividade e boas práticas de marcação.

Como usar:

- Abra o arquivo `index.html` no seu navegador ou hospede o projeto com GitHub Pages.

Contribuição:

- Para melhorias ou correções, abra uma issue ou envie um pull request.

Publicando no GitHub Pages:

- Opção 1 (recomendado): servir do branch `main` (root).
	1. No repositório no GitHub, vá em Settings → Pages.
	2. Em "Source", selecione branch `main` e opção `root` (ou `main` / `/ (root)`).
	3. Salve. A página ficará disponível em `https://SEU_USUARIO.github.io/NOME_DO_REPOSITORIO`.
- Observações:
	- Mantive a estrutura com `index.html` na raiz, a pasta `CSS/` e `img/` na raiz — isso permite ativar o Pages sem mover imagens.
	- Adicionei um arquivo `.nojekyll` para evitar que o GitHub Pages processe arquivos com Jekyll.

Se quiser que eu mova tudo para a pasta `docs/` em vez da raiz (ou crie a branch `gh-pages`), posso fazer isso também.

# Techboard — Landing Page

Este projeto é uma landing page simples desenvolvida usando HTML e CSS.

Breve descrição:

- O projeto contém a página principal em `index.html` e os estilos em `CSS/style.css`.
- A pasta `img/` contém as imagens utilizadas na página.
- Foi desenvolvido como um site estático para apresentação, praticando layout, responsividade e boas práticas de marcação.

Como usar:

- Abra o arquivo `index.html` no seu navegador ou hospede o projeto com GitHub Pages.

Contribuição:

- Para melhorias ou correções, abra uma issue ou envie um pull request.
